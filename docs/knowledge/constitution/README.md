# Knowledge Constitution

The [Knowledge Constitution](knowledge-constitution.md) is the highest normative
authority within CP-MoAKB knowledge governance. It establishes the boundaries
that all Knowledge Authoring Standards (KAS), knowledge policies, authoring
guidance, reviews, and future knowledge work MUST preserve.

The Constitution governs knowledge meaning and governance only. It MUST NOT be
used to define an implementation schema, Runtime behavior, an API, or a physical
record format. Accepted Architecture Decision Records (ADRs), active Runtime
Architecture Specifications (RAS), Design Freeze, Source Policy, and the
Publication Boundary retain their authority within their respective scopes.

KAS-001 applies the Constitution to knowledge-authoring principles. It is not the
Constitution and MUST NOT amend, replace, or silently reinterpret it.
