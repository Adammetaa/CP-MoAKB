# Source Authority Framework

## Purpose and status

This framework governs how future CP-MoAKB datasets assess organizations, publications, authority records, versions, and extraction rights for specific claims. An official publisher is not automatically authoritative for every field, and authority does not make a statement timeless, universal, legally applicable, or true without qualification.

Sprint-015 registers source families and a bounded Rice Department assessment at documentation level only. It does not integrate new sources, copy domain content, authorize extraction, or approve master-data facts.
