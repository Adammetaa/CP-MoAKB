# Platform, Not Diagnosis

CP-MoAKB supplies engineering contracts for governed knowledge infrastructure. It
does not observe a field, infer a pest or disease, choose a treatment, rank
options, calculate economics, or estimate confidence. The HTTP and CLI adapters
expose the same narrow read-only application operations; they do not turn the
library into a decision product.

A consumer considering diagnosis or recommendation must first govern observation,
evidence, scientific interpretation, regulation, safety, uncertainty, human
oversight, and product responsibility. Current APIs are not a shortcut around
those decisions.
