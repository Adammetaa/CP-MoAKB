# Mission and Scope

CP-MoAKB's mission is to provide explainable agricultural knowledge
infrastructure in which identity, authority, evidence, provenance, transformation,
and compatibility remain explicit. The implemented scope is a deterministic,
read-only Python platform plus a separately retained IRAC source pipeline.

The platform supports domain values, constrained adaptation, mechanical
validation, in-memory registries, filtering, structured explanations, JSON
projection, application composition, injected transports, packaging, and
repository governance. It bundles no usable knowledge base and makes no diagnosis,
recommendation, efficacy, safety, or production-deployment claim.
