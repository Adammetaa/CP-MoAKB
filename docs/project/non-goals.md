# Non-goals

Current non-goals include agricultural diagnosis, recommendation, pesticide
selection, ranking, confidence scoring, causality, economic calculation,
scientific inference, AI/LLM integration, a production web service, database-backed
Runtime, automatic file/data discovery, remote fetching, plugins, authentication,
authorization, telemetry, and deployment operations.

The repository is also not a complete agricultural knowledge corpus, regulatory
service, safety authority, or replacement for agronomists and qualified reviewers.
Future documents may explore concepts, but conceptual presence does not make a
capability implemented or approved.
