# Architecture Book

This book describes implemented boundaries only. Start with the
[system overview](system-overview.md), then follow
[layers](layers.md), [runtime flow](runtime-flow.md), and
[dependency direction](dependency-direction.md).

- [Data and control boundaries](data-and-control-boundaries.md)
- [Error boundaries](error-boundaries.md)
- [Security boundaries](security-boundaries.md)
- [Version boundaries](version-boundaries.md)
- [Extension boundaries](extension-boundaries.md)

Normative rules live in the [RAS index](../runtime/specifications/README.md);
durable decisions live in the [ADR index](../ARCHITECTURE_DECISIONS/README.md).
This book explains those authorities and does not supersede them.
