## Draft summary

CP-MoAKB is Explainable Agricultural Knowledge Infrastructure: a governed,
deterministic, read-only Python platform for representing, validating, querying,
explaining, and projecting caller-supplied records.

The candidate includes immutable Runtime models, constrained YAML input,
validation, registries, read-only query and explanation, canonical JSON,
application composition, an injected minimal HTTP adapter, a library-first CLI,
documentation, fictional offline examples, and deterministic release checks.

Package `0.1.0` supports Python 3.11 and 3.12. Installation would use separately
verified attached artifacts only after owner-approved upload; this draft does not
claim package-index availability.

Known limitations: this is an alpha engineering platform, not production-ready,
not a production server, and not a diagnosis, recommendation, ranking,
confidence-scoring, AI, or LLM system. No usable agricultural corpus or official
reference publication is bundled.

Verification covers static analysis, tests, security/documentation/example
contracts, repeated builds, archive contents, and isolated installations. Report
security concerns through [the private reporting policy](../../SECURITY.md).
Repository source is licensed under [Apache-2.0](../../LICENSE). Detailed notes
are in [release notes 0.1.0](release-notes-0.1.0.md).
